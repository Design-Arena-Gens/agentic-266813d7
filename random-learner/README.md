## Random Learner Academy Website

This repository contains the marketing website for **Random Learner Academy**, built with Next.js 15 (App Router) and Tailwind CSS. It showcases programs, learning resources, admissions details, and community events for the academy.

### Local development

Run the development server:

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

### Available pages

- `/` – Home with hero, pathways, events, testimonials, and CTA
- `/courses` – Cohorts, sprints, and partnership learning options
- `/content` – Learning hub with articles, toolkits, and podcasts
- `/admissions` – Admissions steps, tuition plans, and scholarships
- `/about` – Mission, values, team, and impact snapshots
- `/events` – Upcoming gatherings and local chapter details
- `/contact` – Inquiry form and team contact channels
- `/faq` – Frequently asked questions
- `/policies` – Community guidelines and privacy commitments
- `/careers` – Open roles and hiring commitments
- `/content/archive` – Archive of past resources
- `/content/toolkits` – Downloadable facilitation resources

### Production build

```bash
npm run build
npm run start
```
